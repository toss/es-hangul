# getJungseong

단어에서 중성을 추출합니다. (예: `사과` -> `'ㅏㅘ'`)

자세한 예시는 아래 Example을 참고하세요.

```typescript
function getJungseong(
  // 중성을 추출할 한글 문자열
  word: string
): string;
```

## Examples

```tsx
getJungseong('사과');      // 'ㅏㅘ'
getJungseong('하늘');      // 'ㅏㅡ'
getJungseong('띄어 쓰기'); // 'ㅢㅓ ㅡㅣ'
```
