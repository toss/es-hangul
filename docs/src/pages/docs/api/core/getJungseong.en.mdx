# getJungseong

Extracts the Jungseong (vowels) from a Korean word. (Example: 사과 -> 'ㅏㅘ')

```typescript
function getJungseong(
  // Korean string from which to extract the jungseong
  word: string
): string;
```

## Examples

```tsx
getJungseong('사과');      // 'ㅏㅘ'
getJungseong('하늘');      // 'ㅏㅡ'
getJungseong('띄어 쓰기'); // 'ㅢㅓ ㅡㅣ'
```
