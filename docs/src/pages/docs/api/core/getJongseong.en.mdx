# getJongseong

Extracts the Jongseong (final consonants) from a Korean word. (Example: 별 -> 'ㄹ')

```typescript
function getJongseong(
  // Korean string from which to extract the jongseong
  word: string
): string;
```

## Examples

```tsx
getJongseong('별');        // 'ㄹ'
getJongseong('하늘');      // 'ㄹ'
getJongseong('띄어 쓰기'); // ' '
getJongseong('파이팅');    // 'ㅇ'
```
